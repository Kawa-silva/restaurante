# 🍽️ Sistema Web para Restaurante

## 📖 Sobre o Projeto

Este projeto consiste em um sistema web desenvolvido utilizando **Python** e **Django**, com o objetivo de apresentar um restaurante por meio de uma interface moderna e responsiva.

A aplicação permite que os usuários visualizem informações sobre o restaurante, conheçam o cardápio e naveguem entre as páginas do sistema de forma simples e intuitiva.

O projeto também foi utilizado como estudo prático da disciplina de **Gerência de Configuração**, aplicando conceitos de versionamento utilizando **Git** e **GitHub**.

---

# 🚀 Tecnologias Utilizadas

* Python
* Django
* HTML5
* CSS3
* Bootstrap
* JavaScript
* Git
* GitHub

---

# 📂 Estrutura do Projeto

```text
restaurante/
│
├── manage.py
├── db.sqlite3
├── templates/
├── static/
├── setup/
└── README.md
```

---

# ⚙️ Como Executar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/Kawa-silva/restaurante.git
```

### 2. Entre na pasta

```bash
cd restaurante
```

### 3. Instale as dependências

```bash
python -m pip install django
python -m pip install python-dotenv
```

### 4. Execute o servidor

```bash
python manage.py runserver
```

### 5. Abra no navegador

```
http://127.0.0.1:8000/
```

* Navegação entre páginas.
* Organização utilizando Django.
